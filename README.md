# Fuel Efficiency Predictor

This project uses a machine learning model trained on the Auto MPG dataset to predict fuel efficiency (MPG) from vehicle attributes.

## Run locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the app:
   ```bash
   streamlit run app.py
   ```

## Features

- Input widgets for cylinders, displacement, horsepower, weight, acceleration, model year, origin, and car name
- Predicts MPG instantly when the user clicks Predict
